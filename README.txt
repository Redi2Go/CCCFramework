Verwendung:
Es sind zwei Demos vorbereitet: Main.java, MainExample.java
Das UI zeigt einen Graphen an, welcher mit der Zeit modifiziert wird. Die Animation kann mittels den Start-Knopf gestartet werden.
Im Nummern-Feld rechts vom Knopf kann die Anzahl an Millisekunden eingestellt werden, welche nach jeder Aktion gewartet wird.

Disclaimer:
Die Demo der 'Main.java' Klasse führt automatisch jeden Level-Input nach der Reihe durch, d.h. beim Beenden der Animation
öffnet sich direkt der nächste Input in einem neuen Fenster.
